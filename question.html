1. What is dom and what is the purpose of dom?

Document object Model (DOM) is the programming interface for the html document.We can manupulate html element with the help of dom.By the help of dom we can change the html element,attributes,css style,create html events and also able to add and remove the html attributes in the page.

2.How will you select HTML elements using DOM? Name the DOM methods.

There are 5 ways to select html elements using DOM.

  • getElementByTagName()
  • getElementById()
  • getElementsByClassName()
  • querySelector()
  • querySelectorAll()

3. What is event bubble?

Event Bubbling is a type of DOM propagation where the event initially propagate the the innermost target element and gradually it started to propagate the parent event until it reaches the outermost DOM element.

4. What is a callback function and why will you use it?

Callback functions are functions that are called after the first function completes its task.They are often used to handle asynchronous events and make your code more readable. This can make your code more modular and understandable
